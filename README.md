# 📊 End-to-End Retail Sales & Profit Analysis (Python)

## 🔍 Overview
This project analyzes a retail dataset to extract meaningful business insights related to sales, profit, customer behavior, and discount impact.

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ⚙️ Workflow
- Data Cleaning & Preprocessing  
- Feature Engineering  
- Exploratory Data Analysis (EDA)  
- Visualization  
- Business Insights  

## 📊 Key Insights
- Some sub-categories generate losses despite high sales  
- Higher discounts reduce overall profit  
- A small group of customers contributes most revenue  
- Sales performance varies across regions  

## 📁 Project File
- `retail_sales_analysis.ipynb`

## 🎯 Outcome
This project demonstrates how raw data can be transformed into actionable business insights using Python.
